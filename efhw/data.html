首页 / 数据方法论

把每一条接收报文，
当作一次免费的远场探针

天线的方向图通常要在微波暗室或开阔测试场里测。但 PSK Reporter 上每天有几十万条 FT8 接收报告——每一条都记录了"谁、在哪、用多强的信号收到了你"。把这些报文按方位和距离归类，就能在不爬塔、不买仪器的前提下，反推出你的天线在真实电离层下的方向图投影。这一页讲清楚这套方法能测什么、不能测什么。

The Idea

自然界的远场测试场

专业的天线方向图测量，需要在远场把一个已知功率的探针绕着天线转一圈，记录每个角度的场强。PSK Reporter 的数据，本质上提供了一个分布式的、永远在线的等效装置：

  • 全球 7 万+ 接收台分布在各个方位和距离上，相当于布满四周的场强计；
  • 每条报文带有双方网格坐标，两点之间的大圆方位和距离可以纯几何算出；
  • 上报的 SNR 是对端解码器测得的信号强度，大样本平均后能反映相对强弱；
  • FT8 是固定功率、固定调制的标准信号，等效于一个标定过的探针。

把你被收到的所有报文，按方位角分箱、累加信号强度，就得到一张"哪个方向被听得最清楚"的图——这就是你的天线方向图在真实传播条件下的投影。它不是暗室里的纯天线方向图，而是天线 × 电离层 × 对端分布共同作用的结果。理解这一点，是正确解读数据的前提。

What's Solid, What's Modeled

先分清：哪些是硬数据，哪些是模型估计

这套方法的严谨性，取决于诚实区分"几何算出来的事实"和"靠物理模型推出来的估计"。我们对每一类量都标注它的可信度。

实测/几何 直接由坐标与上报值得出，无模型假设 模型推导 经物理模型换算，作趋势参考 待核实 受样本量或外部因素影响，谨慎对待
量来源可信度说明
方位角两端网格 → 大圆方位实测/几何纯几何计算，不含任何假设
距离两端网格 → 大圆距离实测/几何纯几何计算
SNR对端解码器上报准实测受对端天线/噪声影响，大样本平均后稳定
相对 ΔSNR本台 − 同网格邻居实测/几何差分抵消传播共模，天线对比的可靠基准
辐射仰角距离 → 球面单跳模型反推模型推导单跳假设，多跳/混合模式会有偏差，作趋势看
一条贯穿全站的原则：凡涉及"方向、方位、相对强弱"的结论，建立在几何硬数据上，可信度较高；凡涉及"绝对仰角度数"，是球面模型反推的估计值，我们只当趋势参考，不作为精确真值。在线分析看板的二维方向图也在图注里诚实标注了这一点。
The Key Trick

为什么"对邻居差分"是关键

单看自己的 SNR 其实没有意义——今天 20m 条件好，明天差，变化几乎全是电离层在动，跟天线无关。怎么把天线的贡献从传播里剥出来？

答案是差分：取同一时段、同一网格（因此面对几乎相同的传播条件）的邻居台作基准，算你和他们的 SNR 差值 ΔSNR。传播是两者的共模量，相减时被抵消，剩下的就主要是天线、馈线、架设高度和本地噪声环境的差异。

绝对 SNR = 天线 + 馈线 + 传播 + 对端 + 噪声 + 当天条件
邻居 SNR = 邻居天线 + 邻居馈线 + 传播 + 对端 + 噪声 + 当天条件
─────────────────────────────────────────────────────
ΔSNR    = (你的天线系统) − (邻居天线系统)     ← 传播被差掉了

这也顺带解决了另一个陷阱——人口密度偏差。某个方向报文特别多，往往不是天线在那方向强，而是那个方向接收台密集（例如东亚的 FT8 用户极多）。邻居和你面对的是同一批接收台，差分后这个偏差同样被抵消。所以"对邻居的 ΔSNR"比"自己的报文数峰值"更能反映天线本身。

Six Views

六个观测维度

在线分析看板把上述方法拆成几个互补的视角，每个回答一个具体问题。

🧭

主瓣方位与定向性

把每条报文当成一个方位上的单位向量，按圆形统计累加。合向量的方向是主瓣指向，长度（0–1）是方向集中度。方位是圆形量，350° 与 10° 相邻，不能用算术平均——这一点在实现里特别处理过。

📐

二维方向图（方位 × 仰角）

方位（几何硬数据）作角向、仰角（模型估计）作径向，叠成极坐标热力图，颜色是报文密度。直观看出能量集中在低仰角（利远 DX）还是高仰角（利近距）。

📊

逐波段辐射仰角

每个波段辐射仰角的分布（中位、p10、p90），观察多波段天线在各波段是否都保持了低仰角能力。仰角为模型估计，作趋势参考。

⚡

电离层跳距分布

按距离分箱（地波 / 单跳 / 双跳 / 多跳），看能量在不同传播模式下的分布。对比 TX 与 RX 两个方向的远距离占比，可以观察收发是否对称。

🎯

方向 × 波段弱点

对同网格邻居逐（波段 × 方向）算 ΔSNR，排出系统性偏弱的组合。这是把"哪里该改进"落到具体方位和波段的核心视图。

🔇

方向性噪声底限

每个方位上能听到的最弱信号，反映本地噪声环境是否安静、有没有方向性噪声源。它解释了接收端的健康度。

The Loop

从观测到改进的闭环

这套方法真正的价值，不在于"分析完看个热闹"，而在于它能驱动真实的改造，并用同一把尺子量出改造有没有用。

① 理论/几何预测  ──→  ② 数据里的可观测特征
                              │
                              ▼
                        ③ 用对应视图观测
                              │
                              ▼
⑥ 改造后复测同一视图  ←──  ④ 找到偏弱的方向/波段
   看 ΔSNR 是否改善         │
        ▲                   ▼
        └──────────  ⑤ 针对性改造（调向 / 加线 / 改高度）

下一页用一副真实的天线（BG1SB 的 20m 端馈）完整走一遍这个闭环：先纯手算几何预测，再拉数据观测，看两者能不能对上，最后给出有数据支撑的改进方向。
